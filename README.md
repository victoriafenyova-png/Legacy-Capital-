## О проекте

Legacy Capital — это программная платформа для:
- мониторинга инвестиционных портфелей;
- анализа рыночных трендов;
- автоматизации финансовых расчётов;
- генерации отчётности.

**Целевая аудитория:**
- частные инвесторы;
- финансовые консультанты;
- управляющие компании.

## Функции

- **Дашборд** — визуализация ключевых показателей портфеля.
- **Аналитика** — графики доходности, риск‑профиль, корреляции активов.
- **Импорт данных** — интеграция с брокерами и банками (API).
- **Отчётность** — PDF‑экспорт, шаблоны для инвесторов.
- **Уведомления** — алерты по ключевым событиям рынка.
